# 🤖 CYBERWARE MOD - ERWEITERTE FEATURES

## ✨ Massive Expansion Features

### 🎯 1. SLOT-SYSTEM (Neu!)

#### 6 Cyberware-Kategorien mit separaten Slots:

```
┌─────────────────────────────────────────────┐
│ OPERATIVE WARFARE (3 Slots)                 │
│ ✓ Sandevistan ✓ Berserk ✓ [LEER]           │
│                                              │
│ NETRUNNING (2 Slots)                        │
│ ✓ Quickhacking ✓ [LEER]                     │
│                                              │
│ STEALTH SUITE (2 Slots)                     │
│ ✓ Optical Camo [LEER]                       │
│                                              │
│ DEFENSIVE SYSTEMS (2 Slots)                 │
│ ✓ Chrome Body [LEER]                        │
│                                              │
│ SENSORY ENHANCEMENT (3 Slots)               │
│ ✓ Thermal Vision ✓ Ping [LEER]              │
│                                              │
│ ICONIC PACKAGES (1 Slot)                    │
│ [LEER] (NUR EINE!)                          │
│                                              │
│ Chrome Level: ███████░░░ 73%                │
└─────────────────────────────────────────────┘
```

#### Slot-Limits:
- **Operative**: 3 Slots (Kampf-Cyberware)
- **Netrunning**: 2 Slots (Hacking-Fähigkeiten)
- **Stealth**: 2 Slots (Versteck/Unsichtbarkeit)
- **Defense**: 2 Slots (Schutz/Rüstung)
- **Sensory**: 3 Slots (Scanner/Sinne)
- **Iconic**: 1 Slot (Legendäre Cyberware)

**Total: 13 Slots möglich**

#### Chrome-Level System:
- Basiert auf % der belegten Slots
- 0% = Keine Cyberware
- 100% = Vollständig modifiziert (13/13 Slots belegt)
- ⚠️ Höheres Chrome-Level = Mehr Psychose-Risiko (speziell David!)
